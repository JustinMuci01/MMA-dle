MMA WEB GUESSING GAME !!
PLAY HERE: https://fastidious-frangollo-be84b9.netlify.app/


Guess from the current ranked roster of the UFC for the mystery target fighter.
New fighters daily!

TO RUN LOCALLY:

BACKEND -> 
   - Change into 'Python Fetch' directory
   - Ensure Python as well as requirements in requirements.txt are installed
   - Change cors.middleware in main.py to be local machine ip and port
   - fastapi dev main.py will run fastapi app

FRONTEND ->
   - npm run dev

WEB SCRAPING ->
   - All of the web scraping to build the sqlite file is done in builder.py within the 'Python Fetch' folder, using the beautifulsoup and requests libraries
